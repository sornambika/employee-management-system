# employee-management-system
Employee Management System developed using Java with OOP concepts, supporting CRUD operations, search functionality, and efficient data handling using collections.

## 📌 Overview
This project is a simple Employee Management System developed using Core Java. It allows users to manage employee records efficiently by performing operations such as adding, updating, deleting, and viewing employee details.

## 🚀 Features
- Add new employee details
- View all employees
- Update employee information
- Delete employee records
- Search employees by ID or name

## 🛠️ Technologies Used
- Java (Core Java)
- OOP Concepts (Encapsulation, Abstraction)
- Java Collections (ArrayList)
- Exception Handling

## 💡 Project Structure
- `Employee.java` – Defines employee attributes and methods
- `EmployeeService.java` – Contains business logic
- `Main.java` – Entry point of the application

## ▶️ How to Run
1. Clone the repository
2. Open in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run `Main.java`
